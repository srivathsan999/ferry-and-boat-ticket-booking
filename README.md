# Ferry & Boat Ticket Booking Template

## Overview
This is a commercial-grade HTML5/CSS3 template for a Ferry & Boat Ticket Booking website. It is designed with a modern "Ocean Blue" aesthetic, utilizing glassmorphism and a clean, travel-focused layout.

## Tech Stack
- **HTML5**: Semantic markup.
- **CSS3**: Custom properties, Glassmorphism, Responsive design.
- **Bootstrap 5**: Used via CDN for grid and layout utilities.
- **Vanilla JavaScript**: For theme toggling and UI interactions.
- **GSAP**: For smooth animations (via CDN).

## Setup
1.  Open `index.html` in any modern web browser.
2.  No build step required (this is a static template).

## File Structure
- `index.html`: Main Landing Page.
- `home-2.html`: Variant of Home Page.
- `routes.html`: Routes listing with filters.
- `route-details.html`: Single route details with itinerary tabs.
- `schedule.html`: Timetable for departures.
- `pricing.html`: Pricing tiers.
- `blog.html`: Travel blog grid.
- `contact.html`: Contact form and info.
- `assets/`: Contains generic CSS, JS, and Images.

## Customization
- **Colors**: Edit `assets/css/style.css` `:root` variables to change the brand colors.
- **Dark Mode**: Managed by `assets/js/theme-toggle.js` and `assets/css/dark.css`.

## Notes
- Images are generated placeholders. For production, replace them with licensed assets.
- External libraries (Bootstrap, GSAP) are loaded via CDN for immediate ease of use.
